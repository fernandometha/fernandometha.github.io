💡 ความหมายของ "This is..."

"This is..." แปลว่า "นี่คือ..." หรือ "สิ่งนี้คือ..."

ใช้สำหรับแนะนำหรือชี้บอกสิ่งของที่อยู่ ใกล้ตัวผู้พูด (ระยะใกล้)

📌 This = สิ่งนี้ (เอกพจน์)

📝 โครงสร้างประโยค

✅ ประโยคบอกเล่า

This is + a/an + คำนาม
  • 🎈 This is a book. (นี่คือหนังสือ)
  • 🎈 This is an apple. (นี่คือแอปเปิ้ล)
  • 🎈 This is my pen. (นี่คือปากกาของฉัน)

🎯 การใช้ a / an

🔤 ใช้ "a"

หน้าคำนามที่ขึ้นต้นด้วยพยัญชนะ หรือเสียงพยัญชนะ

a book, a cat, a university (เสียง ยู)

🔤 ใช้ "an"

หน้าคำนามที่ขึ้นต้นด้วยสระ (a, e, i, o, u) หรือเสียงสระ

an apple, an egg, an hour (เสียง ออ)

❓ ประโยคคำถาม

🗣️ การถาม-ตอบ

Is this + a/an + คำนาม...?

✅ คำตอบรับ (Yes)

Yes, it is.
(ใช่, มันคือ)

❌ คำตอบปฏิเสธ (No)

No, it isn't.
(ไม่ใช่, มันไม่ใช่)
ตัวอย่าง:
Q: Is this a pencil?
A: Yes, it is. / No, it isn't.

🚫 ประโยคปฏิเสธ

❌ This is not / This isn't

ใช้เมื่อต้องการบอกว่า "สิ่งนี้ไม่ใช่..."

This is not a cat.
หรือ
This isn't a cat.
(สิ่งนี้ไม่ใช่แมว)
💡 ทริค: isn't = is not (รูปย่อ)

🎮 แบบฝึกหัดทบทวน

1. This is ___ apple. (เติม a หรือ an)
a
an
✅ คำตอบที่ถูกต้อง: an
เพราะ apple ขึ้นต้นด้วยเสียงสระ (a) จึงต้องใช้ "an"
This is an apple.
2. Is this a book? (ตอบ Yes หรือ No)
Yes, it is.
No, it is.
✅ คำตอบที่ถูกต้อง: Yes, it is.
การตอบรับต้องตอบว่า "Yes, it is." (ไม่ใช่ No, it is.)
การตอบปฏิเสธคือ "No, it isn't."
3. This is not a cat. = ?
This isn't a cat.
This is a cat.
✅ คำตอบที่ถูกต้อง: This isn't a cat.
isn't = is not (รูปย่อ)
ดังนั้น This is not = This isn't
